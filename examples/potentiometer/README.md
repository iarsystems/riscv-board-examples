# IAR RISC-V GD32V EVAL - Potentiometer example

## Introduction
The potentiometer __R1__ pin is read from the ADC and its value is printed in the stdout on regular intervals.

## Running
The expected stdout output visualization should be similar to the one below, with the values changing when the __R1__ potentiometer position changes:

```
---
Potentiometer example

Value: 1168
Value: 1255
Value: 1347
Value: 1431
Value: 1789
Value: 2247
Value: 3047
...
 ```
